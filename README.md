## Descrição
Classe `Fila` implementa um TDA de fila com 5 níveis de prioridade:
- Muito baixa
- Baixa
- Normal
- Alta
- Muito alta

---

Todos os métodos respeitam a ordem de prioridade:
- `push()` insere elementos
- `pop()` remove elementos da maior prioridade
- `proximoElemento()` retorna o próximo elemento sem remover
- `elemento()` retorna quantidade total
- `vazia()` e `cheia()` retornam booleanos
- `limpar()` esvazia a fila
